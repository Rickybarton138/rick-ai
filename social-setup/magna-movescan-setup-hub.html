Recommended Setup Protocol — Your 4 Brands
🎯 Optimal Account Setup Approach for Astra · Magna · Prime Haul · CoachMentor
Phase 1 — One-time Manual (1 hour per brand): Create the seed Google/email accounts, verify phone numbers, set up personal LinkedIn/Facebook profiles as admins. This is unavoidable and non-automatable.

Phase 2 — Playwright MCP + Claude (already in your stack): Use Claude Code with your Playwright MCP server to automate all profile population — bios, about sections, categories, links, hours. Claude generates all copy, Playwright fills the forms. You just watch and approve.

Phase 3 — Asset Generation (Claude + Canva API): Claude generates all profile descriptions, taglines, and bio copy for each brand across all platforms in one session. Canva API creates all profile images and banners to spec.

Phase 4 — API Token Setup (1 hour): Get long-lived tokens for FB/IG Graph API, LinkedIn, Pinterest, YouTube — these then feed into your n8n workflows permanently. Set once, never touch again.

Phase 5 — n8n Automation Live: Your existing Astra/Magna/MoveScan/CoachMentor n8n workflows take over all posting, DM handling, scheduling, and analytics from this point. Estimated setup time per brand: 2-3 hours total with this stack.
