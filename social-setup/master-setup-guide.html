API Token Quick Reference
Get these in Phase 3
📘 Facebook + Instagram
1
Go to developers.facebook.com/tools/explorer
2
Generate User Token with pages_manage_posts, pages_read_engagement, instagram_basic, instagram_content_publish
3
Exchange for long-lived token (60 day → refresh to never-expire):
GET /oauth/access_token?grant_type=fb_exchange_token&client_id=APP_ID&client_secret=APP_SECRET&fb_exchange_token=SHORT_TOKEN
4
Get Page token: GET /{PAGE_ID}?fields=access_token&access_token={USER_TOKEN}
5
Get IG Account ID: GET /me/accounts → find instagram_business_account → id
▶️ YouTube Data API v3
1
Google Cloud Console → Create project → Enable YouTube Data API v3
2
Credentials → Create OAuth 2.0 Client ID → Web application → Add redirect URI: http://localhost:3000/callback
3
OAuth flow → get authorization code → exchange for tokens → store refresh_token in n8n env vars
4
Refresh token is permanent — use it to get short-lived access tokens in n8n before each API call
🐦 X / Twitter API v2
1
developer.twitter.com → Sign in → Create App
2
App permissions: Read and Write (for posting). Set callback URL.
3
Keys & Tokens tab → Generate Access Token + Secret
4
Free tier = 1,500 posts/month. More than sufficient for CoachMentor's 3 posts/week.
5
Store: API Key, API Secret, Access Token, Access Token Secret → n8n oAuth1Api credential
💼 LinkedIn Company Page
1
linkedin.com/developers → Create App → Associate with company page
2
Request Products: Share on LinkedIn + Sign In with LinkedIn (for posting)
3
OAuth 2.0 → Authorize → Get access token. LinkedIn tokens expire in 60 days — use refresh token flow
4
Get Organization URN: GET /v2/organizationalEntityAcls?q=roleAssignee
📌 Pinterest Business API v5
1
developers.pinterest.com → My Apps → Create App
2
Scopes: boards:read, boards:write, pins:read, pins:write
3
OAuth flow → access token (valid 30 days) + refresh token (long-lived)
4
n8n: store refresh token, exchange for access token at runtime. Excellent API — easiest of all platforms.
🌐 Google Business Profile
1
Same Google Cloud project as YouTube → Enable Business Profile API + My Business Account Management API
2
OAuth 2.0 same credentials as YouTube. Scopes: business.manage
3
Get Location ID: GET /v1/accounts/{accountId}/locations
4
After verification, use POST /v1/{name}/localPosts for GMB posts from n8n
